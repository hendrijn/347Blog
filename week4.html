Toggle Sliders

Since starting my static website I've learned lots of new things like media queries, how to make a profile card, and how to build a nav link area. But I still wanted to include something unique for the blog post so I looked at other fun things to do in CSS. This article by David Walsh covers lots of cool things, but the one that stood out the most to me was the Slide Up and Slide Down function.

How it works is by creating a div with the class of slider filled with random content. Here's a breakdown of the basic slider CSS styling. 

    .slider { 
      overflow-y: hidden;   
      max-height: 500px; /* approximate max height */
      transition-property: all; 
      transition-duration: .5s; 
      transition-timing-function: cubic-bezier(0, 1, 0.5, 1);
    }

Then we set the max height to be 0 on close. I had to look at the source code based on the demo because the website was very unclear on how the rest of the program worked. But I was able to figure it out for the most part, except for the opening animation where it's not as smooth as the closing.
