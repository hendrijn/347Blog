User Login

Javascript and Node.js are still pretty confusing to me. There's a lot there to learn, so I wanted to take some time this week researching easy Node.js projects. It's annoying how much of the internet gives ideas without solutions because it makes me feel a lot better knowing there's code for me to look at and follow along with.

I decided to go to YouTube and find a tutorial. WebDevSimplified, a channel I found over the summer for his CSS battles, has a video for creating a login system with Node.js and several other libraries. I figured this was a perfect project that wouldn't take too much time so I watched and followed along with his video.

In this project, he uses several libraries to aid in the funcionality of the site. (You weren't kidding when you said there would be a lot of them). Despite the obvious Express library, we used nodemon to refresh the server each time we saved the file, passport to keep track of user information throughout their session, bcrypt to encrypt the user's password, and method override to allow the logout button to call the delete function not the patch function. I quickly became familiar with the syntax for importing libraries: 

        const LIB_NAME = require('LIB_NAME')

I also found it interesting how you can render various pages in the project using js and not a simple link tag. For example, when the user visits the url/login, the service renders the login page: 

        app.get('/login', checkNotAuthenticated, (req, res) => {
          res.render('login.ejs')
        })

The project was pretty intense for a newbiebut luckily he offered a step-by-step tutorial. Watching him walk through the process reminded me that scripting is way more complicated than the markup stuff.

To check out the service click here. It starts out on the login page, but if you haven't registered, you'll need to by clicking the register link. After you complete the form, it will take you back to the login page where you enter the same info and login. Now you'll see a welcome message with the name of the account you registered as.
