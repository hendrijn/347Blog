Adding Animations

Getting sick this week has not been fun so I decided to research a topic that's super interesting to me: animating text with CSS. This example was very overwhelming and complex but after looking at it for a while I was able to kind of break down what was happening.

I started with a span representing each letter in the word I wished to animate (so the letters are in-line but targetable). In my case, I wanted to animate "Chalupas" to pay homage to my all-time favorite menu item from Taco Bell. I wrapped all the spans in a div where class="word". This created the structure for my animation, then I added the CSS.

C H A L U P A S

Ok here's the nitty gritty. First I wanted to style the text so I had the color match the white of the screen then set it's size and line-height. I also made it inline block so I got more control over the sizing. Then I used the animation property to add the means to control the movement of the letters. For the spans I put animation: move 5s ease-in-out infinite;. Then I used the @keyframes tag to set different properties of the text at 0%, 50%, and 100% of the animation. Then came targeting each child aka letter. I used the .word span:nth-child(#) selector to delay the animation between each letter. Here's an example of what this looked like for one of the children: 

        .word span:nth-child(6) {
            animation-delay: 2.5s;
          }
