New Beginnings in HTML

It's exciting getting into a class where the main focus is web development. I did a few LinkedIn learning courses over the summer in the 3 main languages, but I still have more to learn. This week was full of knowledge surrounding HTML and it's semantic meaning. The in class labs brought about some questions that I looked up and found helpful resources for. Here's a few of my findings.

This W3Schools article describes how to use HTML links to reference another file in the repository. I needed this to setup the home page because I was unsure how to link the different weeks. It told me to use the relative URL in the href="" attribute. Here's what I used to actually link to this page: 

<a href="/week1.html">

Another thing I needed to research was if there was a date input type. In the lab, Carter and I made a diner menu and wanted the user to input the time at which they were coming in to pick up their order. GeeksforGeeks had a great article describing what the time tag does and how to use it with real code samples. It's cool seeing all the different ways you can ask users for inupt. Here's an example of how we used it in the lab: 

<h3>What time are you coming in?</h3>
<label><input type="time"></label>

which ended up looking like this:

What time are you coming in?
